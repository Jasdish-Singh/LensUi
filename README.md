## ðŸ’» Run it Locally

1.  **Clone the repo**
    ```bash
    git clone [https://github.com/Jasdish-Singh/LensUI.git](https://github.com/Jasdish-Singh/LensUI.git)
    cd LensUI
    ```

2.  **Install packages**
    ```bash
    npm install
    ```

3.  **Add your API Key**
    Create a file named `.env.local` and add your Google Gemini key:
    ```bash
    GEMINI_API_KEY=your_key_here
    ```

4.  **Start the app**
    ```bash
    npm run dev
    ```
